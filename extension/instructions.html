How to Use Stop Shop

Welcome! Stop Shop is a tool designed to help web users limit their time spent on shopping websites.

This page will help you navigate the extension

  • 1: Click set account info, or set time limit.
  • 2: Enter your data, then click submit to store info on local storage.
  • 3: Click start timer
  • 4: You will be notified when 20% of your entered time limit remains. Your shopping tabs will be automatically closed when time is up.

Stop Shop completes database transactions with submitted data to be able to create, read, update, delete stored time limits and user information.
